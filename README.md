# copilot-do-something

A minimal Node.js project with GitHub templates and CI workflow.

## Features

- Node.js 20+ support
- Basic test framework
- GitHub issue and PR templates
- CI workflow with GitHub Actions

## Getting Started

### Prerequisites
- Node.js 20 or higher
- npm

### Installation
```bash
npm install
```

### Running the Application
```bash
npm start
```

### Running Tests
```bash
npm test
```
